# Quantum-Auth Interface

This is a futuristic login page built for a "Cybersecurity" look. It uses advanced CSS and JavaScript to create a cool, high-tech experience.

## Features
- Flow Field Background: Moving green particles using HTML5 Canvas.
- 3D Tilt Effect: The login card moves when you hover your mouse over it.
- Terminal Log: A simulated boot-up sequence that looks like a real hacker tool.
- Glitch Text: The button has a "glitch" animation when hovered.

## Built With
- HTML5 and CSS3
- JavaScript (Canvas API)
- Bootstrap 5
- Vanilla-tilt.js

## Purpose
I built this project to practice Front-end Development and UI Design. It fits my goal of becoming a Cybersecurity professional because it focuses on secure-looking interfaces.
